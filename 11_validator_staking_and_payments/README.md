# Validator Staking & Rewards

## Purpose
This module manages the economic incentives and security deposits for AST validators. It enforces the "Stake-to-Validate" rule, ensuring that all participants have 'skin in the game'. It handles stake locking, reward distribution based on PoT performance, and punitive slashing.

## Core Services & Components
- **Validator Registration**: Onboarding and identity verification.
- **Stake Management**: Locking, unlocking, and freezing of assets.
- **Reward Distribution**: Calculating and issuing epoch rewards.
- **Slashing Engine**: penalizing protocol violations.
