# Validator Node Security Deposit & Payments

## Purpose
This module manages the economic incentives and security deposits for AST validators. It enforces the "PoT Activity + Security Deposit" rule, ensuring that all participants have 'skin in the game'. It handles deposit locking, payment distribution based on PoT performance, and punitive forfeiting.

## Core Services & Components
- **Validator Registration**: Onboarding and identity verification.
- **Deposit Management**: Locking, unlocking, and freezing of assets.
- **Payment Distribution**: Calculating and issuing epoch payments.
- **Forfeiture Engine**: penalizing protocol violations.
