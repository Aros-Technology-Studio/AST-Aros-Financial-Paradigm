# Transaction Simulation Mode

Simulation mode allows testing of complex transactions without affecting state.

## Features

- Runs transactions through full validation and execution logic in sandbox context.
- Provides detailed logs, gas estimates, and compliance feedback.
- Supports batch simulations for scenario analysis and economic modelling.

## Use Cases

- Smart contract developers verifying upgrades.
- Governance evaluating policy changes before implementation.
- Compliance testing new customer onboarding flows.

## Safeguards

Simulation results cannot be promoted to production automatically; manual review required.
