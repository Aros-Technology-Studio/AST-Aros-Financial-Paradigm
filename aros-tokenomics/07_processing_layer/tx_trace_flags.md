# Transaction Trace Flags

Trace flags control diagnostic data captured during processing.

## Flag Types

- **Execution Trace**: Detailed opcode-level tracing.
- **Compliance Trace**: Records compliance checks and outcomes.
- **Performance Trace**: Captures latency and resource usage metrics.
- **Anomaly Trace**: Adds extra logging when risk thresholds exceeded.

## Governance

Trace flag policies protect privacy by limiting default capture. Supervisory override can enable
additional tracing during investigations.
