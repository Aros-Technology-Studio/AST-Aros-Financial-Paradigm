# Crypto Exit Pipeline

Exit pipeline handles conversion of ARO back to external crypto assets.

## Steps

1. Verify user identity and compliance status.
2. Reserve liquidity and select optimal route.
3. Execute swaps or withdrawals on partner networks.
4. Confirm settlement and update audit logs.

## Controls

- Withdrawal limits based on risk profile.
- Cooling periods for large exits.
- Fraud detection monitors unusual patterns.
