# External Crypto Ingestion

External ingestion converts supported cryptocurrencies into ARO via compliant channels.

## Steps

1. KYC verification and wallet whitelisting.
2. Deposit monitoring and confirmation across supported chains.
3. Conversion using liquidity pools or OTC partners.
4. Token issuance into appropriate vault or user account.

## Risk Controls

- Chain analysis tools screen incoming funds.
- Cooling-off periods for large deposits.
- Automated alerts for high-risk jurisdictions.
