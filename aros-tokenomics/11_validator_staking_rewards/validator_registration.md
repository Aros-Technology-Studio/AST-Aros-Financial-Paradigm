# Validator Registration

Registration ensures validators meet technical and compliance criteria before staking.

## Steps

1. Submit application with organisational details and technical specs.
2. Complete compliance review and KYC checks.
3. Deploy validator infrastructure meeting security baselines.
4. Lock required stake and bonds via staking contract.
5. Receive approval and NodeChain credentials.

## Documentation

All registration artefacts stored for audit and linked to governance records.
