# Agent Roles Matrix

Matrix outlines responsibilities for each AI agent.

| Agent | Primary Function | Key Inputs | Outputs |
| --- | --- | --- | --- |
| Validator Monitor | Track validator behaviour and health | Telemetry, PoT scores | Alerts, performance reports |
| Fraud Detector | Identify suspicious transactions | Processing logs, compliance data | Risk scores, escalation tickets |
| Consensus Analyst | Analyse consensus anomalies | NodeChain metrics | Dispute recommendations |
| Liquidity Sentinel | Monitor reserves and velocity | Value circulation telemetry | Buy-back or throttle suggestions |
| Governance Advisor | Model policy impacts | Proposal data, simulations | Impact assessments |
