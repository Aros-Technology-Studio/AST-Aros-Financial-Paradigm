# AST Aros Financial Paradigm Documentation

This folder mirrors the "AST. Aros FINANCIAL PARADIGM and projects" knowledge base that lives in Notion. It captures the complete
multilayered architecture of the Aros Studio tokenomics stack, organised into focused thematic documents. Each file summarises
the related processes, controls, and integration points required to operate the regulated Proof of Transaction economy governed
by AI oversight.

The structure is intentionally modular. Individual teams can work on specific layers while preserving a shared vocabulary
provided by the glossary and change history. Cross-references are kept consistent with the project roadmap and deployment
guidelines so that documents remain actionable for engineering, compliance, and operational stakeholders.

For an overview of the directory layout see the main repository README. The remaining Markdown documents below provide the
full detail imported from Notion.
