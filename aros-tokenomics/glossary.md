# Glossary

Common terminology used across the AST Aros Financial Paradigm.

| Term | Definition |
| --- | --- |
| **ARO** | Native utility token emitted according to Proof of Transaction (PoT) metrics and governed by policy. |
| **PoT** | Proof of Transaction consensus combining behavioural validation with stake and compliance weighting. |
| **NodeChain** | Validator coordination layer responsible for registration, sharding, encryption, and telemetry. |
| **All-Seeing Eye** | Supervisory meta-layer providing holistic oversight, anomaly detection, and escalation workflows. |
| **Compliance Bridge** | Regulated ingress/egress system integrating KYC/AML, liquidity routing, and reporting. |
| **Value Circulation Vault** | Treasury construct managing liquidity pools, reserves, and distribution programmes. |
| **Emission Layer** | Service calculating and executing mint operations using governance-approved policy. |
| **DTE** | Decentralized Transaction Encoding, providing immutable audit records with upgradeable governance. |
| **Supervisory Signal** | Encoded notification (green/amber/red) indicating system health and required actions. |
| **Challenge** | Formal dispute in the PoT engine investigating validator behaviour or transaction validity. |
