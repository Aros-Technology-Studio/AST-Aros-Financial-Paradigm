# Glossary

| Term | Definition |
| --- | --- |
| **ARO** | The native utility token emitted based on validated economic activity through the Proof of Transaction consensus. |
| **PoT** | Proof of Transaction, a consensus protocol that weights validator participation by verified transaction flows. |
| **NodeChain** | The validator coordination network responsible for registration, sharding, encryption, and AI-assisted oversight. |
| **All-Seeing Eye** | Supervisory AI layer that monitors meta-events, anomaly patterns, and escalation procedures across the ecosystem. |
| **Compliance Bridge** | A regulated ingress/egress framework ensuring fiat and crypto flows adhere to KYC/AML requirements. |
| **Meta-Learning Loop** | Feedback pipeline allowing AI agents to refine fraud detection and governance recommendations over time. |
| **Value Circulation Vault** | Treasury construct that routes liquidity, reserves, and distribution schedules for internal and external stakeholders. |
