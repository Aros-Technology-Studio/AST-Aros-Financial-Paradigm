**Intellectual Property Status**

The core architecture of the AST Platform is protected under United States intellectual property law.

Invention Title:** Modular Architecture for Hybrid Transactional Systems**

Application Type: **Provisional Patent Application (under 35 U.S.C. §111(b))**

Filing Date: August 8, 2025

Application Number: 63/860,444

Inventor: Ketevan Arevadze

Scope of Protection

The filed patent covers the novel hybrid framework designed to integrate regulated financial institutions ("fiat operators") with cryptographically-driven digital asset systems ("crypto operators"). Key claims include the Aros Logic Bridge (ALB) protocol, the Governance Rollback Logic, and the Proof-of-Processing consensus integration.
