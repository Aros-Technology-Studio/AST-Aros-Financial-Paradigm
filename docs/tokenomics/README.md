# Tokenomics Documentation

This directory contains key documentation regarding the economic model of the AST/AFC ecosystem.

## Contents

- [Pricing Model](./pricing_model.md): Defines the dynamic price model (`P = alpha * log(U) + beta * FX + gamma`) and emission formulas.
- [Validator Metrics](./validator_metrics.md): Details the Transactional Validator Score (TVS), Node Reputation Index (NRI), and reward distribution mechanics.
- [Cryptographic Proofs](./proofs.md): Explains the Proof of Claim (PoC) and Deterministic Proof Hash (DPH) algorithms used for audit trails and verification.
